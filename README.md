# dev.Svenlaa.com

- **Framework**: [Next.js](https://nextjs.org)
- **Styling**: [Tailwind](https://tailwindcss.org)
- **Deployment**: [Vercel](https://vercel.com)

## Running Locally

This application requires Node.js

```bash
git clone https://github.com/svenlaa/home.git
cd home
yarn
yarn dev
```
